## 2. 確率と確率分布
  |名前|数式または記号|説明|関連|

  |試行||||
  |根元事象||試行によって起こりうる個々の結果<br>素事象、標本点ともいう||
  |全事象|Ω|すべての根元事象の集合、標本空間ともいう||
  |和事象|A<sub>1</sub> ∪ A<sub>2</sub>|少なくともA<sub>1</sub>もしくはA<sub>2</sub>の１つが起こる||
  |積事象|A<sub>1</sub> ∩ A<sub>2</sub>|A<sub>1</sub>とA<sub>2</sub>が同時に起こる||
  |空事象|∅|何も起こらない事象||

  |独立性|P(B\|A) = P(B), P(A\|B) = P(A)<br>P(A ∩ B) = P(A)P(B)|一方の事象が起こるかどうかが他方の事象の起こる確率に影響しないこと<br>二つの確率がある場合でお互いの結果が互いに影響しない場合は**互いに独立**という||
  |排反|<img src="image-5.png" width="300px">|2つの事象AとBが同時に起こらない場合、つまり排反事象の場合の確率||
  |一般的な確率|P(A) = A/Ω|Aは起こった事象の数<br>Ωは全事象<br>古典的な定義でありラプラスの定義ともいう||
  |期待値|E(x) = Σ p<sub>k</sub> × x<sub>k</sub>|pは確率<br>xは取りうる値<br>1回の試行で得られる値の平均値のこと||
  |加法定理 1|P(A ∪ B) = P(A) + P(B)|A ∩ B = ∅の場合|
  |加法定理 2|P(A ∪ B) = P(A) + P(B) - P(A ∩ B)|A ∩ B ≠ ∅の場合|
  |条件付き確率|P(B\|A) = P(A ∩ B) / P(A)<br>P(A ∩ B) = P(B\|A) × P(A)|Aという条件下でBが起こる確率|乗法定理|
  |乗法定理|P(A ∩ B) = P(B\|A) × P(A)||条件付き確率|
  |ベイズの定理|P(H<sub>i</sub>\|A) = P(H<sub>i</sub> ∩ A) / P(A)<br>P(H<sub>i</sub>\|A) = P(H<sub>i</sub>) × P(A\|H<sub>i</sub>) / P(A)|原因から結果を探る手法<br>P(A)は事前確率<br>P(H<sub>i</sub>\|A)は事後確率||
  |離散型確率分布||<img src="image.png" width="300px">|一様分布<br>二項分布<br>多項分布<br>ポアソン分布<br>幾何分布<br>超幾何分布|
  |確率変数||変数Xのことである確率に対して取り得る値||
  |確率関数||f(x<sub>i</sub>)と表わされる<br>離散型の場合は0 ≤ f(x<sub>i</sub>) ≤ 1となり<br>連続型の場合はΣf(x<sub>i</sub>) = 1となる||
  |確率質量関数|f(x) = P(X=x)<br>Σ P(X=x<sub>i</sub>) = 1|すべてを足し合わせると１になる||
  |連続型確率分布|P(X=x) = 1/∞|取りうる値に切れ目がないため、無限となる<br><img src="image-1.png" width="300px">|連続一様分布<br>正規分布<br>指数分布<br>t分布<br>F分布<br>カイ二乗分布|
  |確率密度関数|P(a ≤ X ≤ b) = ∫<sub>a</sub><sup>b</sup> f(x) dx|算出される値は面積を表していて、その値が確率となる<br>また、**1を超えることがある**<br>連続型確率変数ともいう||
  |累積分布関数|離散型<br>F(x) = P(X ≤ x) = Σ P(X)<br><br>連続型<br>F(x) = P(-∞ ≤ X ≤ x) = ∫<sub>-∞</sub><sup>x</sup> f(t) dt|確率変数Xが**ある値x以下(X ≤ x)の値となる確率**<br><br>大文字のFを用いてF(x)と表わす<br><br>離散型<br><img src="image-12.png" width="300px"><br><br>連続型<br><img src="image-13.png" width="300px"><br><br>確率密度関数f ← → 累積分布関数Fと表わされる<br><br>F(∞) = 1<br>確率変数がある範囲の値しかとらないときに、Xがとる値がその範囲の最大値以上となる場合に成立する<br><br>F(-∞) = 0<br>確率変数Xがとる値がマイナス無限大となる確率は0である||
  |期待値|離散型<br>E[X] = Σ x<sub>i</sub> × f(x<sub>i</sub>) = μ<br><br>連続型<br>∫<sub>-∞</sub><sup>∞</sup> x × f(x) dx = μ|平均と同じ意味||
  |分散|離散型<br>V[X] = E[(X-μ)²] = Σ(x<sub>i</sub> - μ)² × f(x<sub>i</sub>) = σ²<br><br>連続型<br>V[X] = E[(X-μ)²] = ∫<sub>-∞</sub><sup>∞</sup> (x-μ)² × f(x) dx = σ²<br><br>また...<br>V[X] = E[X²] - μ²<br>としても定義される|バラつきを表す指標となり標準偏差（σ）の2乗||
  |モーメント|歪度（わいど）<br>n / ((n-1)(n-2)) × Σ((x<sub>i</sub>-x̄)/s)³<br><br>尖度（せんど）<br>n(n+1) / ((n-1)(n-2)(n-3)) × Σ(x<sub>i</sub>-x̄)⁴/s⁴ - 3(n-1)² / ((n-2)(n-3))|歪度と尖度で表される<br><br>歪度（わいど）<br>分布が正規分布からどれだけ歪んでいるかを表す統計量で、左右対称性を示す指標のことです。サンプルサイズをn、各データの平均値をx̄、標準偏差をsとすると歪度は次の式から求められる<br><img src="image-2.png" width="500px"><br><br>尖度（せんど）<br>分布が正規分布からどれだけ尖っているかを表す統計量で、山の尖り度と裾の広がり度を示します。サンプルサイズをn、各データの平均値をx̄、標準偏差をsとすると尖度は次の式から求められます。<br><img src="image-3.png" width="500px">||
  |ベルヌーイ分布|P(X=1) ≡ f(1) = p<br>P(X=0) ≡ f(0) = 1-p<br><br>期待値<br>μ = E[X] = 1 × p + 0 × (1-p) = p<br><br>分散<br>σ² = E[X²] - μ² = 1² × p + 0² × (1-p) - μ² = p(1-p)|ある確率に応じた１か０かの二つの値をとる<br>0 ≤ p ≤ 1の範囲となる||
  |二項分布|P(X=x) = f(x) = <sub>n</sub>C<sub>x</sub> × p<sup>x</sup> × (1-p)<sup>n-x</sup><br><br>期待値<br>E[X] = np<br><br>分散<br>V[X] = np(1-p)|成功率pのn回ベルヌーイ試行を行ったとき、成功回数がxとなる確率<br><br>B(n,p)と記すこともある||
  |ポアソン分布|f(x) = e<sup>-λ</sup> × λ<sup>x</sup> / x!<br><br>期待値<br>E[X] = λ<br><br>分散<br>V[X] = λ|Po(λ)<br>ここでnpを一定の値「λ（ラムダ）」とおき、np = λのままでnを十分大きくpを十分に小さくした場合の二項分布は、平均λのポアソン分布に近似することができます。<br><br>「ある期間に平均λ回起こる現象が、ある期間にX回起きる確率の分布」||
  |幾何分布|P(X=k) = (1-p)<sup>k-1</sup> × p (k = 1,2,3,...)<br><br>期待値<br>E(X) = 1/p<br><br>分散<br>V(X) = (1-p)/p²|成功確率がpである独立なベルヌーイ試行を繰り返す時、初めて成功するまでの試行回数Xが従う分布である。<br>幾何分布の無記憶性、ある事象が発生する確率は、その事象が発生する前の情報の影響を受けない||
  |超幾何分布|P(X=k) = (<sub>M</sub>C<sub>k</sub> × <sub>N-M</sub>C<sub>n-k</sub>) / <sub>N</sub>C<sub>n</sub><br><br>k以下の条件を満たす<br>max(0, n-(N-M)) ≤ k ≤ min(n, M)<br><br>期待値<br>E(X) = n × M/N<br><br>分散<br>V(X) = n × M(N-M)/N² × (N-n)/(N-1)|AとBで構成されるN個からなる集団があり、AがM個、BがN-M個であるとします。この集団から取り出されたn個の中に含まれるAの個数が従う確率分布||
  |負の二項分布|1.<br>P(X=x) = <sub>k+x-1</sub>C<sub>x</sub> × p<sup>k</sup> × (1-p)<sup>x</sup><br>(k = 1,2,3,...)<br><br>期待値<br>E(X) = k × (1-p)/p<br><br>分散<br>V(X) = k × (1-p)/p²<br><br>2.<br>P(X=x) = <sub>x-1</sub>C<sub>k-1</sub> × p<sup>k</sup> × (1-p)<sup>x-k</sup><br>(k = 1,2,3,...)<br><br>期待値<br>E(X) = k/p<br><br>分散<br>V(X) = k × (1-p)/p²|1.<br>成功率がpである独立なベルヌーイ試行を繰り返す時、**k回成功するまでの失敗回数Xが従う分布**<br><br>2.<br>成功率がpである独立なベルヌーイ試行を繰り返す時、**k回成功するまでの試行回数Xが従う分布**||
  |正規分布|f(x) = 1/(√(2πσ²)) × e<sup>-(x-μ)²/(2σ²)</sup><br>(-∞ < x < ∞)<br><br>もしくは<br>f(x) = 1/(√(2πσ²)) × exp(-(x-μ)²/(2σ²))<br>(-∞ < x < ∞)<br><br>期待値<br>E(X) = μ<br><br>分散<br>V(X) = σ²|統計学における検定や推定、モデルの作成など様々な場面で活用される連続型確率分布<br>横軸を確率変数を、縦軸はそのときの確率密度を表す<br><br><img src="image-4.png" width="300px"><br><br>**平均μ、分散σ²に従う正規分布**といい、**X ~ N(μ,σ²)**<br><br>**再生性**<br>正規分布N(μ₁,σ₁²)に従うあるデータと、そのデータとは独立な別の正規分布N(μ₂, σ₂²)に従うデータを足したデータは、正規分布N(μ₁ + μ₂, σ₁²+σ₂²)に従う<br><br>標準正規分布への変換<br>正規分布の中で「平均μ = 0、分散σ² = 1」である正規分布を**標準正規分布**という、標準正規分布はX~N(0,1)と書く||
